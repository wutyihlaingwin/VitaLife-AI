# VitaLife AI

VitaLife AI is a web-based health, nutrition and wellness management system developed as part of the Higher Diploma in Science in Computing final project at the National College of Ireland (NCI).

The application helps users manage their health by creating personalised nutrition plans, exercise recommendations and supplement suggestions based on their personal health profile.

## Technologies Used

- Java 17
- Spring Boot
- Spring MVC
- Thymeleaf
- Spring Data JPA
- MySQL
- Bootstrap 5
- HTML5
- CSS3
- Google Gemini API
- Maven
- Git & GitHub

## Project Features

### Sprint 1
- Project setup
- MySQL database configuration
- User authentication
- Health profile management
- BMI calculation
- BMR calculation
- TDEE calculation
- Initial user interface

### Sprint 2
- Nutrition and meal planner
- Exercise planner
- Supplement recommendation module
- Progress tracking
- Integration testing
- User interface improvements

## Development Methodology

This project follows the Agile Software Development methodology using Scrum. Development is organised into multiple sprints and managed using Trello.
